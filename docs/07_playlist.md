# Phase 1-G: 再生リスト機能（Day 13-14）

## 目的
用件（テーマ・目的）付きの再生リストを作成・閲覧・共有できる機能

## タスク

### 1. DBテーブル追加
- [x] 初期スキーマ（00001_initial_schema.sql）に含まれており作成済み

### 2. 再生リスト作成ページ（`/playlist/new`）
Stitchデザインに基づく:
- [x] タイトル入力
- [x] 用件・テーマ入力（textarea）
- [x] 公開設定ドロップダウン（公開 / 限定公開 / 非公開）
- [x] カバー画像アップロード（任意、または最初の動画サムネイルを自動使用）
- [x] 動画追加セクション:
  - 「+ 動画を追加」ボタン → URL入力 → メタデータ自動取得
  - 追加済み動画のリスト表示
  - 各動画: サムネイル / タイトル / VTuber名 / 種別バッジ / 削除ボタン
  - 各動画にコメント入力フィールド（任意、200文字以内）
- [x] 「作成する」ボタン（グラデーション）
- [x] バリデーション（タイトル必須、最低1本の動画）

### 3. 再生リスト詳細ページ（`/playlist/[id]`）
Stitchデザインに基づく:
- [x] ヒーローセクション（グラデーション背景 + タイトル）
- [x] 「用件」ラベル + 説明文
- [x] 作成者情報（アバター + ユーザー名 + 作成日）
- [x] 統計: 動画数 / いいね数 / ブックマーク数
- [x] アクションボタン: 👍いいね / 🔖保存 / 共有
- [x] 番号付き動画リスト:
  - 番号 / サムネイル / タイトル / VTuber名 / 種別バッジ / 再生時間
  - 作成者のコメント（グレーテキスト）
  - クリックで元URL（YouTube/X）に遷移
- [x] 自分のリストの場合: 削除ボタン

### 4. 再生リスト一覧ページ（`/playlist`）
Stitchデザインに基づく:
- [x] ソート: 人気順 / 新着順
- [x] 再生リストカード（縦リスト）:
  - カバーサムネイル
  - タイトル
  - 用件スニペット（1行、省略表示）
  - 作成者アバター + ユーザー名
  - 動画数 + いいね数
- [x] 「+ 再生リストを作成」ボタン

### 5. いいね・ブックマーク（再生リスト用）
- [x] 再生リスト詳細ページのいいね/ブックマークボタン
- [x] トグル動作 + 楽観的UI更新
- [x] カウンター更新

### 6. マイページ連携
- [ ] マイページに「再生リスト」タブ追加
- [ ] 自分が作成した再生リスト一覧

## 完了条件
- [x] 用件付きの再生リストが作成できる
- [x] 動画の追加・削除が動作する
- [x] 再生リスト詳細ページで動画リスト + コメントが表示される
- [x] 一覧ページでブラウズできる
- [x] いいね/ブックマークが動作する
