# 推しシェア - ユーザー獲得・グロース戦略

---

## 1. ターゲットユーザー分析

### 1.1 VTuberファンのペルソナ

矢野経済研究所の2025年調査（15〜44歳男女37,353名対象）に基づく。

#### ペルソナA：ライト層（新規ファン / 切り抜き勢）
- **年齢層**: 15〜22歳（Z世代中心）
- **性別**: 男女比 6:4
- **行動特性**: 切り抜き動画中心の視聴、タイパ重視、複数VTuberを並行視聴
- **利用SNS**: X(Twitter)、TikTok、YouTube Shorts、Instagram
- **ニーズ**: 「推しの面白い配信をサクッと知りたい」「新しいVTuberを開拓したい」
- **サービス利用動機**: おすすめ配信を手軽に発見、友達への布教ネタ収集

#### ペルソナB：コア層（古参ファン / ガチ恋勢・DD勢）
- **年齢層**: 20〜35歳
- **性別**: 男性やや多め（7:3）
- **行動特性**: ライブ配信をリアルタイム視聴、スパチャ・メンバーシップ加入、グッズ購入
- **利用SNS**: X(Twitter)、Discord、Reddit、5ch/なんJ
- **ニーズ**: 「伝説の配信を記録・共有したい」「推しの魅力を体系的に布教したい」
- **サービス利用動機**: 神回アーカイブ作成、初見向け布教リスト作成

#### ペルソナC：クリエイター層（切り抜き師・まとめ師）
- **年齢層**: 18〜30歳
- **性別**: 男性やや多め
- **行動特性**: 切り抜き動画制作、ファンアート作成、まとめ記事執筆
- **利用SNS**: X(Twitter)、YouTube、Discord、pixiv
- **ニーズ**: 「注目配信の情報収集を効率化したい」「自分の切り抜きの再生数を伸ばしたい」
- **サービス利用動機**: トレンド把握、切り抜きネタ発掘、自コンテンツの宣伝

### 1.2 最もニーズが高いユーザー層

**最優先ターゲット: ペルソナB（コア層）**
- 理由: 推し布教への強いモチベーション、SNS発信力が高い、UGC生成の主力
- 「この配信だけは見て！」リストを作りたい欲求が強い

**次点ターゲット: ペルソナA（ライト層）**
- 理由: ユーザー数のボリュームゾーン、口コミ受信者として重要
- コア層が作ったおすすめリストの消費者

### 1.3 新規ファン vs 古参ファンのニーズ比較

| 観点 | 新規ファン | 古参ファン |
|------|-----------|-----------|
| 求めるもの | 入門ガイド、ベスト配信TOP10 | 隠れた名配信、時系列アーカイブ |
| 共有行動 | 「これ面白い！」と気軽にシェア | 「このVTuberの魅力を語りたい」と体系的に布教 |
| 滞在時間 | 短い（回遊型） | 長い（深掘り型） |
| 課金意欲 | 低い | 高い（推しのためなら出費する） |
| リテンション | 離脱リスク高い | 長期利用が期待できる |

## 2. ユーザー獲得チャネル

### 2.1 Twitter/X（最重要チャネル）

**規模感**: VTuber関連ツイートは毎日数十万件。ファンの主要な情報収集・発信プラットフォーム。

**施策**:
- **ハッシュタグ戦略**: 各VTuberの配信タグ（例: #ぺこらいぶ）に関連するおすすめ投稿を促進
- **OGPカード最適化**: 共有時にVTuberのサムネイル・タイトル・おすすめコメントが表示される魅力的なカードを自動生成
- **インフルエンサー連携**: フォロワー1万人以上のVTuberファンアカウントに早期アクセス提供
- **Botアカウント運用**: 「今日のおすすめ配信」を自動投稿するBotを運用（各事務所・VTuber別）
- **話題性のある投稿設計**: 「〇〇の配信、今週のおすすめランキング」をウィークリーで自動生成・投稿

**KPI**: 月間インプレッション100万回、プロフィールクリック率2%以上

### 2.2 Reddit

**規模感**: r/Hololive 150万メンバー、r/Nijisanji 約10万メンバー、r/VirtualYoutubers 約50万メンバー

**施策**:
- コミュニティのルールを遵守しつつ、有用なコンテンツとしておすすめリストを投稿
- 「Weekly Recommendation Thread（週間おすすめスレ）」の定期投稿を提案
- 英語圏ユーザー向けの機能も視野に入れ、海外展開の足がかりとする
- Reddit AMAやコミュニティイベントへの参加

**注意点**: Redditは露骨な宣伝を嫌う。価値提供ファーストで、サービスURLは控えめに

### 2.3 5ch / なんJ / まとめサイト

**規模感**: VTuber関連スレッドは常時数十が並行稼働。まとめサイト（やらおん、にじさんじまとめ等）は月間数百万PV規模。

**施策**:
- まとめサイト運営者へのAPI提供（おすすめランキングウィジェット）
- 「神回データベース」としてスレッド内で自然に言及される品質のコンテンツ作成
- まとめサイトのSEOの強さを活用したバックリンク獲得

**注意点**: 直接的な宣伝は炎上リスクあり。あくまで「便利なツール」として自然に認知されることを目指す

### 2.4 Discord

**規模感**: Discord全体で月間アクティブユーザー2.59億人（2025年）。VTuber関連サーバーは数千規模で存在。

**施策**:
- ホロライブ・にじさんじ等の大手ファンDiscordサーバーでの口コミ促進
- Discord Bot開発: `/recommend [VTuber名]` でおすすめ配信を返すBot
- 公式Discordサーバー開設: コミュニティ形成とフィードバック収集
- Botの導入実績を増やし、ファンサーバー管理者へのアプローチ

**KPI**: 50サーバー以上へのBot導入、公式Discord 1,000人以上

### 2.5 SEO（中長期チャネル）

**ターゲットキーワード（検索ボリューム推定）**:
- 「[VTuber名] おすすめ配信」（月間1,000〜10,000）
- 「[VTuber名] 神回」（月間1,000〜5,000）
- 「VTuber おすすめ 初心者」（月間500〜3,000）
- 「ホロライブ おすすめ配信」（月間1,000〜5,000）
- 「にじさんじ 面白い配信」（月間500〜3,000）
- 「[VTuber名] 入門」（月間100〜1,000）

**施策**:
- 各VTuber・事務所ごとの「おすすめ配信まとめ」ページを自動/半自動で生成
- 構造化データ（Schema.org）の活用でリッチスニペット表示を狙う
- ロングテールキーワード戦略: 「〇〇 マイクラ 神回」等の具体的なクエリに対応
- UGCによる自然なページ増加でドメインパワーを強化

**KPI**: 6ヶ月で月間オーガニック流入5万PV

## 3. バイラル設計

### 3.1 OGPデザイン戦略

**コンセプト**: 「共有したくなるカード」= 推し布教カード

**デザイン要素**:
```
┌─────────────────────────────────┐
│  [VTuberサムネイル画像]          │
│                                  │
│  ★★★★★ 4.8（128人が推薦）      │
│  「〇〇の伝説のホラー配信」      │
│                                  │
│  推薦コメント:                   │
│  「初見さんに絶対見てほしい神回」 │
│                                  │
│  🎬 推しシェア で詳細を見る      │
└─────────────────────────────────┘
```

**ポイント**:
- VTuberのサムネイルを大きく表示（視覚的インパクト）
- 星評価と推薦人数で「みんなが認めた神回」感を演出
- ユーザーの推薦コメントを表示（UGC感で信頼性向上）
- X(Twitter)の大きいカード形式に最適化（1200x630px）

### 3.2 「推し布教」文化の活用

VTuberファンの「推しを広めたい」欲求をサービス設計に組み込む。

**布教リスト機能**:
- 「〇〇入門セット」: 推しVTuberのおすすめ配信を5〜10本まとめたリストを作成可能
- 「布教プレイリスト」として1つのURLで共有可能
- 「この布教リストで〇人が新規ファンになりました」的なフィードバック

**布教バッジ・実績システム**:
- 「布教マスター」: 作成したリストが100回以上閲覧
- 「神キュレーター」: 推薦した配信の平均評価4.5以上
- SNSプロフィールに貼れるバッジ画像の自動生成

**共有導線の最適化**:
- 配信おすすめ登録後に「Xで共有」ボタンを目立つ位置に配置
- 共有テンプレート: 「[VTuber名]の[配信タイトル]が神回すぎた！ #VTuberおすすめ [URL]」
- 定型文は編集可能にして、ユーザーの個性を出せるようにする

### 3.3 ランキング・トレンド機能

**リアルタイム性**:
- 「今週の神回ランキング」: 直近1週間で最も推薦された配信
- 「急上昇おすすめ」: 短期間で推薦数が急増した配信
- 「殿堂入り配信」: 累計推薦数トップの不朽の名作

**パーソナライズ**:
- 「あなたの推しの新着おすすめ」通知
- 「似た趣味のファンが推している配信」レコメンド
- フォロー中のキュレーターの新着おすすめ

**競争要素**:
- 月間キュレーターランキング（最も支持された推薦者）
- VTuber別「布教貢献度」ランキング
- 事務所別・グループ別のトレンド比較

## 4. コミュニティ施策

### 4.1 切り抜き師とのコラボ

**価値提案**: 切り抜き師にとって「どの配信を切り抜くべきか」の情報源となる

**施策**:
- 切り抜き師向けダッシュボード: トレンド配信・注目シーンのタイムスタンプ情報
- 切り抜き動画へのリンク機能: おすすめ配信ページに関連する切り抜き動画を紐づけ
- 切り抜き師ランキング: 良質な切り抜きを制作するクリエイターの可視化
- 切り抜き師への収益分配: アフィリエイトプログラムの提供（将来的に）

### 4.2 VTuber本人・公式との関係構築

**段階的アプローチ**:
1. **Phase 1（受動的）**: サービスがファンの間で自然に広まるのを待つ。VTuber本人のコンテンツを尊重し、著作権に配慮した設計
2. **Phase 2（認知）**: VTuber本人やマネージャーにサービスの存在を認知してもらう。ファンからの自発的な紹介を促進
3. **Phase 3（協力）**: VTuber本人が「おすすめ配信」を自ら選定する公式機能を提供。事務所との正式な連携協議

**注意点**:
- VTuberの権利を最優先に考えた設計（配信リンクのみ、転載しない）
- 事務所のガイドラインを厳守
- ネガティブなコンテンツ（低評価・批判）は扱わない方針を明確化

### 4.3 UGC促進の仕組み

**投稿のハードルを下げる**:
- ワンクリックで「おすすめ」登録（URLと一言コメントだけでOK）
- テンプレートコメント提供: 「泣ける」「笑える」「歌が最高」等のタグ選択式
- 配信URLを貼るだけでサムネイル・タイトル・配信者情報を自動取得

**投稿のモチベーションを上げる**:
- いいね・ありがとう機能: 「この推薦のおかげでいい配信に出会えた」フィードバック
- キュレーターポイント: 活動に応じたポイント付与（将来的な特典と連動）
- 週間/月間MVPキュレーター表彰
- 「あなたの推薦で〇人がこの配信を見ました」通知

## 5. KPI設計と目標数値

### 5.1 収益目標からの逆算

**前提条件（広告収益モデル）**:
- Google AdSense RPM: 300〜400円/1,000PV（エンタメ系サイト平均）
- アフィリエイト（YouTube Premium等）: 追加収益源
- 1PVあたり単価: 約0.3〜0.5円（広告＋アフィ合算）

**月10万円達成に必要な規模**:
| 収益源 | 単価想定 | 必要規模 |
|--------|---------|---------|
| AdSense（RPM 350円） | 0.35円/PV | 月間28.6万PV |
| アフィリエイト込み（0.5円/PV） | 0.5円/PV | 月間20万PV |

**月20万円達成に必要な規模**:
| 収益源 | 単価想定 | 必要規模 |
|--------|---------|---------|
| AdSense＋アフィ（0.5円/PV） | 0.5円/PV | 月間40万PV |

**PVからDAU/MAUへの換算**:
- 1ユーザーあたり平均PV: 3〜5PV/セッション
- 月間セッション数: 平均2〜3回/月
- → 月間20万PV ≒ MAU 2〜3万人、DAU 2,000〜3,000人
- → 月間40万PV ≒ MAU 4〜6万人、DAU 4,000〜6,000人

### 5.2 KPI一覧

| KPI | 定義 | 目標値（月10万円時） |
|-----|------|-------------------|
| MAU | 月間アクティブユーザー | 25,000〜30,000 |
| DAU | 日間アクティブユーザー | 2,500〜3,000 |
| 月間PV | ページビュー | 200,000〜300,000 |
| DAU/MAU比率 | エンゲージメント指標 | 10〜15%（コンテンツサイトとして標準的） |
| 平均セッション時間 | 滞在時間 | 3分以上 |
| おすすめ投稿数/月 | UGC量 | 5,000件以上 |
| SNS共有数/月 | バイラル指標 | 2,000回以上 |
| オーガニック流入比率 | SEO健全性 | 40%以上 |

### 5.3 フェーズ別成長マイルストーン

#### Phase 0: プレローンチ（ローンチ前 1ヶ月）
- **目標**: コアユーザー100人のベータテスター確保
- **施策**: Xでのティザー投稿、VTuberファンDiscordでの告知
- **KPI**: ウェイティングリスト登録500人

#### Phase 1: ローンチ〜1ヶ月目
- **MAU目標**: 1,000〜2,000
- **月間PV**: 5,000〜15,000
- **月間収益目標**: 2,000〜5,000円
- **重点施策**:
  - Xでの集中的な告知（VTuberファンインフルエンサー5〜10人に依頼）
  - r/Hololive、r/Nijisanji への投稿
  - Discord Bot の初期リリース
  - コアユーザーからのフィードバック収集と改善
- **成功基準**: 週次リテンション率 30%以上

#### Phase 2: 3ヶ月目
- **MAU目標**: 5,000〜8,000
- **月間PV**: 30,000〜60,000
- **月間収益目標**: 1〜2万円
- **重点施策**:
  - SEO本格始動（主要VTuber名 × おすすめ配信のページ群を構築）
  - 切り抜き師コミュニティとの連携開始
  - ランキング・トレンド機能のリリース
  - まとめサイトへのウィジェット提供
- **成功基準**: 月次リテンション率 25%以上、オーガニック流入比率20%以上

#### Phase 3: 6ヶ月目
- **MAU目標**: 15,000〜25,000
- **月間PV**: 100,000〜200,000
- **月間収益目標**: 5〜10万円
- **重点施策**:
  - パーソナライズレコメンド機能の実装
  - Discord Bot の大規模展開（50サーバー以上）
  - アフィリエイト収益の本格化
  - コミュニティイベント（おすすめ配信コンテスト等）
- **成功基準**: DAU/MAU比率 12%以上、SNS共有数1,000回/月以上

#### Phase 4: 1年目
- **MAU目標**: 30,000〜50,000
- **月間PV**: 250,000〜500,000
- **月間収益目標**: 10〜20万円
- **重点施策**:
  - 英語圏への展開（ホロライブEN、Phase Connect等）
  - VTuber事務所との公式連携交渉
  - プレミアム機能の検討（広告非表示、高度な分析等）
  - APIの外部公開
- **成功基準**: 複数の収益源確立、コミュニティの自走化

## 6. リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| VTuber事務所からの差し止め | 高 | 配信リンクのみ扱い転載しない。ガイドライン厳守。事前に二次創作ガイドラインを確認 |
| 競合サービスの出現 | 中 | コミュニティの早期構築で先行優位。UGCの蓄積がモート（堀） |
| ファンコミュニティでの炎上 | 中 | 宣伝色を出さない。価値提供ファースト。ネガティブコンテンツ排除 |
| SEO順位の変動 | 中 | 複数チャネルへの分散。SNS・コミュニティ経由の直接流入確保 |
| ユーザー離れ（飽き） | 中 | 継続的な機能アップデート、季節イベント、新VTuberへの対応 |
